class  GestionProduits
 

Description détaillée

La classe GestionProduits. Cette classe permet de gérer les produits en les récupérant depuis la base de donées. Ces produits seront ensuite affichés dans la page principale avec une limitation de 3 produits, et dans la page dédié aux produits.
Elle permet aussi de recupérer les informations de chacun de ces produits. Ces informations seront ensuite affiché dans la page de détails associé au produit.
